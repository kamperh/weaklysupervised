Weakly supervised
=================

A theme based on [Lanyon](https://lanyon.getpoole.com/).


Install
-------
Install Ruby 2.5+:

    sudo apt-get install ruby ruby-dev
    ruby -v

Install bundler:

    sudo gem install bundler

Install required packages:

    bundle install


Serve
-----
Serve locally at <http://localhost:4000/>:

    bundle exec jekyll serve
